# 太閣立志傳V 威力加強版工具

太閣立志傳V 威力加強版工具：  

https://my.feishu.cn/docx/TbqqdAyuxoQl73x5arScMDkVntb

本 repo 目前主要整理「事件文字、劇情腳本、對話分支、襲擊事件、溫泉事件」等文字資源。  

圖片素材尚未納入，之後可再規劃上傳位置或用 AI 生成補齊。

## 專案概覽

本專案以 `function/` 下的 `.txt` 檔案為主，依照事件類型與劇情用途分類整理，方便後續修改、校對、擴寫與維護。

目前內容包含：

- 場景入口與事件入口
- 個人戰劇情
- 人妻主線與共用訊息
- 溫泉事件
- 商店／日常場景劇情
- 強盜／襲擊事件
- 醫師事件
- 妻子互動與對話
- 個人戰攔截事件

## 相關 MOD

另外有製作一個立繪替換 MOD，可搭配本 repo 的豔遇事件使用：

- Patreon：https://www.patreon.com/posts/139533660
- 內容：將賊、忍者、海盜、商人、劍豪與其他 NPC 的立繪替換為女性圖片。
- 說明：因 Patreon 的 R18 內容只能放在收費頁面，認識的人可以直接找我要；不認識的話就花 3 美元支持一下。
