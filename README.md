## 專案概覽

本專案以 `function/` 下的 `.txt` 檔案為主，依照事件類型與劇情用途分類整理，方便後續修改、校對、擴寫與維護。

目前內容包含：

- 場景入口與事件入口
- 個人戰劇情
- 人妻主線與共用訊息
- 溫泉事件
- 商店／日常場景劇情
- 強盜／襲擊事件
- 醫師事件
- 妻子互動與對話
- 個人戰攔截事件
